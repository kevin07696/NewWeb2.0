Introduction to Pointers


#include <iostream>

int main() {
	int b = 10;
	int *a = &b;
	std::cout << "The address location of b = " << &b << std::endl;
	std::cout << "The content at the pointer a = "<< a << std::endl;
	std::cout << "a points to the content at b = "<< *a << std::endl;
	return 0;
}

Let's Learn About Memory

Ask permission from the computer to allocate memory space to store a datatype (in this case int). Name the allocated memory b.

int b;

By assigning a value to b, we store the integer 10 at the memory location we call b.

b = 10;

Ask the computer to read the content at b. Next, add that content to the integer 1. Then, I am asking for the summation to be assigned to b. Notice that 1 is not a double like 1.0. Since b only has enough memory for integer values, adding other datatypes like std::string, double, or long int can cause unpredictable errors.

b = b + 1;
