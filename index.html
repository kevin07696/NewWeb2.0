I was so excited because of the recent public interest in cryptocurrency and blockchain capabilities. Blockchain is a more secure iteration of what is essentially linked lists. You take an element from the list and use a node to store the hash or address of the next element. Thus, if there is any tampering of one element in the linked lists, the hash would change breaking the links of hash codes.
