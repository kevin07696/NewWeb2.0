An Introduction to Recursion

What is Recursion?

Recursion is used for looping a solution. A "while loop" or a "for loop" would be iterative, while recursion is the opposite of that. To implement recursion you first need to have a strong understanding of functions. Then, search in your problem for the end of your loop or when your problem is at its most basic form. This will be how you stop your function. Next, formulate the subroutine. A subroutine is just an expression that returns a part of the solution.

Let's begin by analyzing the recursive program for finding factorials:


#include <iostream>
using namespace std;

int f(int n){

	if (n <= 1)
		return 1;
	else 
		return n*f(n-1);
}
int main(){
	int num;
	cout<< "Enter a number: ";
	cin>>num;
	cout<< "Factorial of entered number: "<< f(num);
	return 0;
}
